    Formation Rules

    Regular Verbs

    -ER Verbs: Drop -er → -é
    (e.g., parler → parlé)

    -IR Verbs: Drop -ir → -i
    (e.g., finir → fini)

    -RE Verbs: Drop -re → -u
    (e.g., vendre → vendu)

    Verbs using Être

    A small group of verbs, mostly involving movement or state change, use être. Remember the mnemonic DR MRS VANDERTRAMP.

    Irregular Verbs & Exceptions

    Many common verbs are irregular. Some follow patterns:

    Ending in -is:
    prendre → pris
    mettre → mis

    Ending in -it:
    dire → dit
    écrire → écrit

    Ending in -u / -û:
    voir → vu
    pouvoir → pu
    devoir → dû
    lire → lu

    Other common irregulars:
    faire → fait
    être → été
    avoir → eu
    mourir → mort
